# The Open Institute for Professional Judgment

**The umbrella over everything this project measures, certifies, and publishes.** A repo ships features; an institute holds standards. This charter names what already operates here and binds it to a public cadence.

## What the Institute operates

| Organ | What it is | Cadence |
|---|---|---|
| **[SkillSpec](../SKILLSPEC.md)** | The conformance standard for agent skills (L1→L3), with the [validator](../skillspec/) and [live badge](../skillspec/README.md#the-badge) | RFC process, versioned |
| **[The Operator's Exam](https://mohitagw15856.github.io/pm-claude-skills/certified.html)** | The open, verifiable credential for professional judgment | Always open; item bank reviewed quarterly |
| **[SkillBench](../skillbench/)** | The professional-work benchmark for models ([public rankings](https://mohitagw15856.github.io/pm-claude-skills/modelbench.html)) | Frozen task sets, versioned; runs published with full transcripts |
| **[The Census](reports/)** | The measurement of the public skill ecosystem | Weekly automated; flagship annually |
| **[The Season](https://mohitagw15856.github.io/pm-claude-skills/season.html)** | The global competition — identical hidden state, attested scores | Monthly; Hall of Seasons permanent |
| **[The Annual Report](reports/)** | The State of Professional AI — the citable synthesis | Yearly (mid-year edition inaugural) |

## Principles (the ones that make the marks worth anything)

1. **Verifiability over authority** — every credential, score, and ranking is cryptographically checkable or reproducibly derived. We publish methods before results.
2. **Money never touches judgment** — sponsorship buys naming, never placement in rankings, exam outcomes, census findings, or the curated library. (See [SPONSORSHIP.md](SPONSORSHIP.md) house rules.)
3. **Open corpus, open instruments** — the skills, the validators, the benches, the exam machinery: MIT, forkable, auditable.
4. **Honest limits, stated in the artifact** — every instrument names what it cannot measure, in the same document as its results.

## Governance (v1, honest about scale)

Today the Institute is one maintainer and a public process: changes to instruments happen by PR with the reasoning in the open; disputes are argued in issues; the standing decisions live in [OPERATIONS.md](../OPERATIONS.md). The intent as adoption grows: SkillSpec spins into a [neutral foundation](FOUNDATION.md) with co-maintainers from adopting projects. Written down now so the trajectory is a commitment, not a pivot.

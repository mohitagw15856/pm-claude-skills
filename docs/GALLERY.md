# 📸 The Gallery — the creative side of a skills library

The [README](../README.md) stays clean and skills-first on purpose — this page is where the visual experiments live. Every screenshot links to the real, running thing.

> All of these are static pages in [`web/`](../web/) — no backend, no accounts. They exist because software should occasionally be fun, and because each one is also a different *doorway into the same 599 skills*.

# PM Skills for Obsidian

Two commands: **Insert skill instructions at cursor** (the full SKILL.md body, ready for Copilot-for-Obsidian / Text Generator / any AI plugin) and **Insert skill reference link**. 520+ skills, fetched from the public index, cached per session.

## Submit to the community list (owner)
1. Test locally: copy `manifest.json` + `main.js` into `<vault>/.obsidian/plugins/pm-skills/`, enable in settings.
2. Create repo `mohitagw15856/obsidian-pm-skills` with these files + a release tagged `0.1.0` (attach main.js + manifest.json as release assets).
3. PR to obsidianmd/obsidian-releases adding the plugin to `community-plugins.json` (id `pm-skills`). Review takes ~1–3 weeks.

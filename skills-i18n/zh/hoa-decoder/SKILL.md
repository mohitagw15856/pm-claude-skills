# HOA 解读技能

买进一个HOA，等于加入一个对你拥有征税权的微型政府。这个技能像一位在业委会会议上坐过很多次的朋友那样读CC&Rs：哪些规则会改变你的生活方式、意外账单能有多大、以及该先索要哪些记录。

## 这个技能产出什么

- 使用限制条款的解读，按这位买家的生活方式影响排序
- 特别摊派敞口：业委会能征收什么、需要什么投票、上限是什么
- 执行与罚款机制——违规如何升级，直至留置权
- 出租限制解读、费用上涨问题清单，以及要索取的记录

## 所需输入

仅在未提供时索取：

- **文件**——CC&Rs、章程、规则、费用/预算页。解读已提供的内容；储备金研究、预算和会议纪要通常不在CC&Rs里——把它们标为待索取记录。
- **他们打算怎么生活**——宠物、车辆、居家办公、将来出租、装修改造。排序取决于这些。
- **当前月费和已知摊派**（若文本中没有）。

## 框架：严重程度分级

- 🔴 **会让你损失真金白银**——无上限或低门槛的特别摊派、会累积并可能演变为留置权的罚款（某些地方甚至可强制拍卖——依司法辖区而定）、扼杀投资计划的出租禁令/配额、标准含糊的改造审批、无上限的月费上涨、过户费。
- 🟡 **不寻常——买前深究**——与买家声明计划相冲突的限制（宠物、车辆、居家办公、装饰）、无决定期限的建筑审查、业委会无需业主投票即可修改规则的权力。
- 🟢 **标准**——常规的滋扰、垃圾和公共区域规则；标注出来，别让买家被模板吓到。

按以下主线通读文件：**使用限制**、**摊派机制**（月费上涨上限；批准门槛；未写上限 = 警示）、**出租限制**（配额、排队、最短租期、既有权利保留）、**执行链条**（通知 → 听证 → 罚款 → 留置；逐步引用原文）、**修改规则**（规则多容易在你脚下改变）。当条款的可执行性存疑时，标注*"可执行性因司法辖区而异——请本地核实"*，而不是宣布其无效。

## 输出格式

### HOA 解读：[社区名称]

**1. 结论**——放心买 / 睁着眼买 / 这个HOA会和你的生活方式作对——三句话。

**2. 按对你的影响排序的限制条款**

| 限制（§） | 条款怎么写 | 如何冲击你的计划 | 严重程度 |
|---|---|---|---|

**3. 💸 金钱敞口**——今天的月费、上涨机制、特别摊派规则引用、现实的最坏账单；罚款表和留置路径。

**4. 🚩 警示信号，分级排序**——引用的措辞、咬人的情景、严重程度。

**5. 买前要索取的记录**——储备金研究（及其充足率）、24个月业委会纪要、预算和欠费率、总保单、过去/未决的特别摊派与诉讼、出租配额现状和排队情况。

**6. 问业委会/物业经理的问题**——费用上涨历史（"过去5年每年的月费是多少？"）、即将到来的大修、罚款实际执行的频率。

在产出物结尾原文附上：*"这是一份大白话解读，不构成法律/财务建议——法律因司法辖区而异；任何关键事项请向合格专业人士确认。"*

## 质量检查

- [ ] 限制条款按这位买家声明的生活方式排序，而非文件顺序
- [ ] 特别摊派机制被引用；"未写上限"本身即为 🔴 信号
- [ ] 执行链条从实际文本逐步追溯
- [ ] 待索取记录清单具体，并说明每项为何重要
- [ ] 未提供的文件（储备金研究、纪要、预算）被点名为缺口
- [ ] 免责声明逐字出现在产出物中

## 反模式

- [ ] 不要虚构文件中不存在的限制或权力
- [ ] 不要为显得客观而软化警示信号——无上限的摊派权就是一张空白支票
- [ ] 不要把依赖司法辖区的规则（留置/拍卖权、可执行性）说成普适
- [ ] 不要按规则听起来多严格排序——按它是否触及这位买家的生活排序
- [ ] 不要跳过财务健康问题——只看CC&Rs永远看不出一个破产的HOA

## 依据

买方侧HOA尽职调查实践——CC&R分诊、摊派敞口分析、记录清单。

# 拒赔解读技能

拒赔信写出来是为了终结对话；申诉之所以存在，是因为对话往往不该终结。相当一部分拒赔——尤其是编码错误、"非医疗必需"和材料缺失——在有人用证据而非愤怒回应*所引理由*时被推翻。这个技能解读拒赔信到底主张了什么，把每项主张配上能回应它的证据，然后写出那封读起来像出自一个会继续升级的人之手的申诉信。

## 这个技能产出什么

- 拒赔解读：所引理由的大白话版本，以及该理由类型对申诉胜算意味着什么
- 与拒赔类型匹配的证据清单——动笔前先收集什么
- 申诉信——逐点回应、逐项引用文件、紧扣期限
- 升级阶梯——内部申诉层级、外部/独立复核、监管投诉——按顺序排列

## 所需输入

仅在未提供时索取：

- **拒赔信文本**——所引理由代码/措辞，及声明的申诉期限。还没有信？第一步是要求书面拒赔及具体保单依据。
- **理赔经过**——申报了什么、何时、损失或治疗情况、任何事前授权或理算员往来。
- **保单条款**（如有）——申诉信最有力的句子是拿保单打拒赔。
- **已提交过什么**——以及保险公司声称从未收到什么（经典桥段）。

## 框架：严重程度分级

读出拒赔信所引理由并分类：

- 🔴 **通常可回应——带证据申诉**——"非医疗必需"（用主治医生信函+病历+指南回应）、编码/计费错误（由医疗方更正代码回应）、"未收到材料"（重新提交并留递送凭证回应）、无证据的"既往状况"或"既有损坏"断言（用有日期的记录回应）、压价估值（用独立估价回应——这是争议不是拒赔，信里应重新定性）。
- 🟡 **更难但可争**——"实验性/研究性"治疗（需要指南和同行评议支持）、有正当理由的逾期申报、保单措辞含糊的除外条款解释（模糊通常不利于起草方——标注依司法辖区而定，值得一争）。
- 🟢 **大概率站得住的拒赔**——明确除外的风险/治疗、损失日时保单已失效、保单期外的理赔；诚实说出来——无望的申诉耗掉的是其他可行选项（协商、分期）的期限。

申诉的结构永远相同：**复述拒赔信原话 → 用所附证据具体回应该理由 → 引用支持理赔的保单条款 → 提出具体请求 → 点明所处期限及未获答复时的下一步。**绝不去争信里没引的理由——那等于教他们新的拒赔理由。

## 输出格式

### 拒赔解读与申诉：[理赔编号，保险公司]

**1. 解读**——所引理由的大白话、类型归属、诚实的胜算框架（可回应/可争/大概率站得住）。

**2. 证据清单**

| 拒赔断言 | 回应它的证据 | 从哪获取 | 状态 |
|---|---|---|---|

**3. 申诉信**——全文可发送：理赔信息抬头 · "贵司[日期]函称：'[引用]'" · 附证据编号的逐点回应 · 保单条款引用 · 具体请求 · 可追踪寄送说明。

**4. 升级阶梯**——内部申诉层级及期限 → 外部/独立复核权利（依计划和司法辖区而定——读拒赔信自己的权利告知部分）→ 监管投诉 → 律师函/请律师的门槛。

**5. 期限盒**——信中和保单里每个要紧的日期。

在产出物结尾原文附上：*"这是一份大白话解读，不构成法律/财务建议——法律因司法辖区而异；任何关键事项请向合格专业人士确认。"*

## 质量检查

- [ ] 申诉逐字复述并回应拒赔信的原话
- [ ] 申诉信中每项断言都指向一份附件
- [ ] 胜算框架诚实——无望的拒赔被点名并给出替代路径
- [ ] 升级阶梯注明哪些层级依司法辖区/计划而定
- [ ] 所有期限提取进一个盒子，寄送日期建议在期限内
- [ ] 免责声明逐字出现在产出物中

## 反模式

- [ ] 不要写愤怒信——理算员把怒气排到队尾；证据才推动案卷
- [ ] 不要去争信里没引的理由——只回应所主张的
- [ ] 不要把翻案概率说成百分比——按理由类型的模式呈现，不编统计
- [ ] 不要为收集完美证据而错过期限——按时合格提交，之后补充
- [ ] 不要在估值争议时漏掉重新定性——"拒赔"和"压价"是两套打法

## 依据

投保人申诉实践——拒赔理由分诊、证据匹配、逐点申诉起草、升级排序。

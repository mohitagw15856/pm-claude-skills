# HOA 解読スキル

HOA に買い込むことは、あなたに課税権を持つ小さな政府に加入することだ。このスキルは、理事会の会議に何度も座ってきた友人のように CC&Rs を読む：どのルールがあなたの暮らし方を変えるか、想定外の請求書はどこまで大きくなり得るか、どの記録を最初に要求すべきか。

## このスキルが産出するもの

- この買い手のライフスタイルへの影響順にランク付けした使用制限の解読
- 特別徴収のエクスポージャー：理事会は何を、どの投票で、どの上限のもとで課せるか
- 執行と罰金のメカニズム——違反がどうエスカレートし、リーエン（担保権）まで至るか
- 賃貸制限の解読、費用上昇の質問リスト、請求すべき記録

## 必要な入力

提供されていない場合のみ求める：

- **書類**——CC&Rs、細則、ルール、費用/予算ページ。提供された分を解読する。修繕積立調査、予算、議事録は通常 CC&Rs にはない——請求すべき記録としてフラグを立てる。
- **どう暮らすつもりか**——ペット、車両、自宅ビジネス、将来の賃貸、リフォーム。ランク付けはこれ次第。
- **現在の管理費と既知の徴収**（本文にない場合）。

## フレームワーク：深刻度スケール

- 🔴 **実際にお金を失わせ得る**——上限なし・低い承認基準の特別徴収、累積してリーエンに発展し得る罰金（一部の法域では競売まで——法域依存）、投資計画を殺す賃貸禁止/枠、基準の曖昧な改修承認、上限のない管理費値上げ、譲渡手数料。
- 🟡 **異例——購入前に掘り下げる**——この買い手の表明した計画と衝突する制限（ペット、車両、自宅ビジネス、装飾）、決定期限のない建築審査、組合員投票なしにルールを変更できる理事会権限。
- 🟢 **標準**——通常の迷惑行為、ゴミ、共用部のルール。定型文に買い手が怯えないようラベルを付ける。

書類を次の観点で通読する：**使用制限**、**徴収メカニズム**（値上げ上限。承認基準。上限の記載なし = フラグ）、**賃貸制限**（枠、順番待ち、最短賃貸期間、既得権）、**執行の連鎖**（通知 → 聴聞 → 罰金 → リーエン。各段階を引用）、**改正ルール**（ルールがあなたの足元でどれほど簡単に変わり得るか）。執行可能性が疑わしい条項には、無効と宣言する代わりに *「執行可能性は法域により異なる——現地で確認を」* と印を付ける。

## 出力形式

### HOA 解読：[コミュニティ名]

**1. 評決**——安心して買える / 目を開けて買う / この HOA はあなたのライフスタイルと戦う——三文で。

**2. あなたへの影響順の制限**

| 制限（§） | 記載内容 | あなたの計画への打撃 | 深刻度 |
|---|---|---|---|

**3. 💸 金銭エクスポージャー**——現在の管理費、値上げメカニズム、特別徴収ルールの引用、現実的な最悪の請求書。罰金表とリーエンへの道筋。

**4. 🚩 危険信号、ランク順**——引用した文言、牙をむくシナリオ、深刻度。

**5. 購入前に請求する記録**——修繕積立調査（と積立率）、24か月分の理事会議事録、予算と滞納率、マスター保険証券、過去/係争中の特別徴収と訴訟、賃貸枠の現状と順番待ち。

**6. 理事会/管理会社への質問**——値上げ履歴（「過去5年の各年の管理費は？」）、近く予定される大規模修繕、罰金が実際に課される頻度。

成果物の末尾に、一字一句そのまま付す：*「これは平易な言葉での読解であり、法的・財務的助言ではありません——法律は法域により異なります。重要事項は資格を持つ専門家に確認してください。」*

## 品質チェック

- [ ] 制限がこの買い手の表明したライフスタイル順にランク付けされ、書類順ではない
- [ ] 特別徴収メカニズムが引用され、「上限の記載なし」自体が 🔴 のフラグ
- [ ] 執行の連鎖が実際の文言から段階ごとに追跡されている
- [ ] 請求すべき記録リストが具体的で、各項目の理由付き
- [ ] 未提供の書類（修繕積立調査、議事録、予算）がギャップとして名指しされている
- [ ] 免責文が一字一句そのまま成果物に現れる

## アンチパターン

- [ ] 書類にない制限や権限を創作しない
- [ ] バランスを装って危険信号を和らげない——上限なしの徴収権は白紙の小切手だ
- [ ] 法域依存のルール（リーエン/競売権、執行可能性）を普遍として提示しない
- [ ] ルールの厳しそうな響きでランク付けしない——この買い手の生活に触れるかどうかでランク付けする
- [ ] 財務健全性の質問を省かない——CC&Rs だけでは破産寸前の HOA は見えない

## 基盤

買い手側の HOA デューデリジェンス実務——CC&R トリアージ、徴収エクスポージャー分析、記録チェックリスト。
